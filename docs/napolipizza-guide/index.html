        LOVII PIZZA - Руководство по проекту

        Выполненные задачи

        1. Структура проекта

        • Создан полный HTML/CSS/JS проект
        • Настроен Docker + Docker Compose конфигурация
        • Создан Nginx конфигурационный файл
        • Добавлены скрипты для запуска (Unix/Windows)

        2. Дизайн и UX

        • Премиум дизайн в стиле Michelin 2-Star
        • Итальянская цветовая схема (красный, зеленый, белый акценты)
        • Типографика: Playfair Display (заголовки) + Lato (текст)
        • Адаптивный дизайн для всех устройств
        • Красивые градиентные заглушки вместо изображений

        3. Функциональность

        • Мобильное бургер-меню
        • Интерактивное меню с вкладками (Пицца, Паста, Салаты, Десерты, Напитки)
        • Форма бронирования столика с валидацией
        • Плавная прокрутка к секциям
        • Подсветка активного раздела навигации
        • Параллакс эффект для hero-фона
        • Lightbox для галереи
        • Ripple эффект на кнопках
        • Анимация счетчика опыта

        4. Контент

        • Hero секция с призывом к действию
        • Секция "О нас" с особенностями ресторана
        • Полное меню с ценами (в EUR)
        • Галерея с placeholder изображениями
        • Форма бронирования с полями даты/времени
        • Контактная информация
        • Footer с навигацией и соцсетями

        Текущий статус

        Сайт запущен и доступен по адресу: http://localhost:8000

        Сервер работает через Python HTTP Server в фоновом режиме.

        Следующие шаги

        Для демонстрации заказчику:

        1. Добавить изображения (опционально, но рекомендуется): - Поместить фото в src/images/ согласно README.md - Или использовать текущие градиентные заглушки (выглядят стильно!)

        2. Настроить контент: - Отредактировать цены в меню (сейчас в EUR) - Обновить контактную информацию - Добавить реальные ссылки на соцсети

        3. Настроить backend для форм: - Форма бронирования сейчас показывает alert - Нужно подключить отправку на email или в CRM

        4. Deploy: - Загрузить на GitHub Pages - Или настроить production сервер

        Дизайнерские решения

        Цветовая палитра

        • Итальянский красный (#c41e3a) - акценты, кнопки, цены
        • Глубокий синий (#2c3e50) - заголовки, текст
        • Золотой акцент (#f39c12) - детали
        • Теплый белый (#faf8f5) - фон

        Типографика

        • Playfair Display - элегантные заголовки с засечками
        • Lato - чистый, читаемый текст без засечек

        Пространство

        • Section padding: 100px вертикально
        • Container width: 1200px максимум
        • Щедрые отступы между элементами

        Анимации

        • Плавные переходы (0.3s - 0.6s)
        • Transform для производительности
        • Intersection Observer для fade-in эффектов
