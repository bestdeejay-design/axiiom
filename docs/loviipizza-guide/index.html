        LOVII PIZZA — Руководство по проекту

        Обзор проекта

        LOVII PIZZA — современный сайт доставки неаполитанской пиццы с красивым и функциональным дизайном. Двухколоночный layout со sticky-навигацией, корзиной, системой лояльности и мотивации, профилями доставки и бэкендом на Yandex Cloud Functions.

        • 333 товара в 18 категориях
        • Пицца 30 см, Piccolo 20 см, Кальцоне
        • Роллы, суши, хлеб, фокачча
        • Комбо-наборы, десерты, напитки
        • Замороженная продукция, ароматные масла
        • Мастер-классы, франшиза, контакты

        Архитектура

        Frontend

        • index.html — двухколоночный layout (sidebar + контент)
        • css/main.css — все стили
        • js/config.js — конфигурация приложения
        • js/app-delivery.js (~1420 строк) — вся логика: загрузка меню, корзина, сортировка, отправка заказа

        Backend

        Go-функция на Yandex Cloud Functions — тонкий транспорт к Max Bot API. Принимает сформированный на фронте HTML заказа и форвардит в Telegram-канал.

        • backend/max-order/index.go — handler (CORS, валидация, форвард)
        • backend/setup.sh — bootstrap (SA + Lockbox)
        • backend/deploy.sh — деплой версии функции
        • Секреты (MAX_TOKEN, MAX_CHAT_ID) — в Yandex Lockbox
        • Rate limit: 30 req/min на инстанс
        • CORS whitelist: pizza.lovii.ru + локальные dev-порты

        Тайм-менеджмент

        Автоматическое переключение режимов работы по времени:

        • 10:00–20:30 — обычный режим, приём заказов
        • 20:30–21:00 — предупреждение «Доставляем до 22:00»
        • 21:00–22:00 — предупреждение + запрос адреса (самовывоз)
        • 22:00–06:00 — закрыто, ссылка на Яндекс Еду

        Конфигурация

        Все настройки в js/config.js:

        • ORDER_CONFIG.maxEndpoint — URL Yandex Cloud Function
        • ORDER_SOURCES — справочник источников трафика
        • SOURCE_CONFIGS — профили доставки (home/partner)
        • CATEGORY_GROUPS — группы категорий для ленивой загрузки
        • CATEGORY_MAP — русские названия категорий
        • HIT_IDS — ID товаров-хитов
        • MAINTENANCE_MODE — заглушка «временно не работаем»
        • LAZY_LOAD_CONFIG — настройки ленивой загрузки
