# CONTEXT_MAP.md — Источник Истины проекта LOVII

## Бизнес-контекст

**Продукт:** LOVII — цифровая инфраструктура локальной экономики. SaaS-платформа для микробизнеса (оператор лояльности, цифровая витрина, City Franchise).

**Ценностное предложение:** Продукт создаётся на базе 20-летнего опыта аутсорс-компании ООО «Аксиома». Это не стартап с нуля, а выделенный product-юнит с финансовой подушкой.

**Целевая аудитория:** Инвесторы в ЦФА (цифровые финансовые активы). Минимальный порог — 1 000 ₽.

## Технический стек

- **Frontend:** Vanilla HTML/CSS/JS, без фреймворков
- **Шрифты:** Inter (UI), Noto Serif (документы)
- **Темы:** Light/Dark через CSS-переменные (data-theme), localStorage
- **PWA:** manifest.json + service-worker.js + иконки 192/512px
- **Иконки:** Только SVG (Feather-стиль, stroke-width: 1.8)
- **Хостинг:** GitHub Pages (bestdeejay-design.github.io/lovii_presentation)
- **Домен:** invest.lovii.ru (custom domain)
- **Репозиторий:** github.com/bestdeejay-design/lovii_presentation
