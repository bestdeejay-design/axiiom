ФУНКЦИОНАЛ ДЕМО-ВЕРСИИ PADELPRO (актуально на июнь 2026)
========================================================

СТАРТОВАЯ СТРАНИЦА (#/)
-----------------------
- Тёмный градиентный херо (navy → синий) с бейджем «SaaS для падел-клубов»
- Крупный заголовок PadelPro с градиентным акцентом на «Pro»
- 3 карточки преимуществ с иконками: P&L в реальном времени, Честный корт, Авторасчёт зарплат
- 3 карточки входа с демо-подсказками (Клиент / Тренер / Администратор)
- Тёмный футер: цена 50 000 ₽/мес, пробный период 1–2 месяца бесплатно
- Адаптив: карточки в столбик на мобильных


МОДУЛЬ КЛИЕНТА (мобильное приложение)
=====================================

Вход (#/login/client)
---------------------
- 5 карточек клиентов: имя, телефон, тип абонемента, остаток (цветной бейдж)
- Ручной ввод номера с маской +7 (XXX) XXX-XX-XX
- Валидация на blur (11 цифр), Enter = вход

Главный экран (#/client)
-------------------------
- Абонемент-бар: тип, остаток, прогресс-бар, срок действия
- Кнопка «Пополнить» → модалка: 4 занятия / 8 000 ₽, 8 / 14 000 ₽ (−12%), 12 / 19 000 ₽ (−20%)
- Вкладки: Предстоящие / История
- Предстоящие: карточки тренировок (время, дата, тренер, корт, цена, статус-бейдж)
  - Отмена с модальным подтверждением + Toast
  - Повторить (rebook) на завершённых
- История: группировка по месяцам с раскрытием (Месяц — N тренировок — сумма)
  - Каждая запись: дата, время, тренер, корт, статус, цена
- Bottom nav: Тренировки / Запись / Сканер / Профиль

Запись (#/client/book)
----------------------
- 5 шагов: Дата (7 дней) → Время (сетка 4×4, занятые серые) → Тренер (свободен/занят) → Корт (фото, тип, цена) → Подтверждение
- Слоты показывают число свободных тренеров и кортов
- Цена = стоимость корта (indoor 2000 ₽, outdoor 1500 ₽)

Сканер QR (#/client/scan)
--------------------------
- Камера с прицельной рамкой, авто-распознавание jsQR
- Успех: списание 1 занятия, Toast
- Ошибки: чужая запись, не начата, уже подтверждена
- Ручной ввод ID записи
- Сканер останавливается при уходе с экрана

Профиль (#/client/profile)
---------------------------
- Аватар, имя, телефон
- Кнопка «Редактировать» → модалка: имя, телефон, email
- Статистика: тип абонемента, остаток (цветной), всего посещений, срок действия


МОДУЛЬ ТРЕНЕРА (мобильное приложение)
=====================================

Вход (#/login/trainer)
----------------------
- 6 карточек тренеров: имя, специализация, ставка, код (бейдж)
- Ручной ввод кода, Enter = вход

Главный экран (#/trainer)
--------------------------
- Статистика месяца: часы, заработано (₽), ставка (₽/ч)
- Активная сессия: оранжевая полоса с пульсацией, живой таймер, кнопка завершить
- Вкладки: Сегодня / Завтра / Неделя (список карточек)
- Карточка: время, клиент, корт, статус-бейдж
- «Начать» → QR-модалка с таймером 5 мин, кнопка «Обновить QR»
- «Завершить» → расчёт зарплаты, Toast
- Алерт если completed без подтверждения клиента
- Bottom nav: Расписание / Зарплата / Профиль

Зарплата (#/trainer/salary)
----------------------------
- Зелёная карточка: начислено за месяц
- Группировка по дням с раскрытием (день — N тренировок — сумма)
- Выбор месяца через выпадающий список
- CSV-экспорт ведомости

Профиль (#/trainer/profile)
----------------------------
- Имя, код, специализация
- Статистика: заработано за месяц, часов, ставка, всего тренировок, дата найма


МОДУЛЬ АДМИНИСТРАТОРА (десктоп с сайдбаром)
============================================

Вход (#/login/admin)
--------------------
- Логин/пароль (admin / admin123), презаполнены, Enter = вход

Сайдбар
-------
- Баннер «ДЕМО-ВЕРСИЯ»
- ← На главную
- Секции: Основное / Управление / Аналитика
- Пункты: Дашборд, Сетка, Тренировки, Тренеры, Клиенты, Корты, Финансы, Реестр
- Хлебные крошки на всех страницах

Дашборд (#/admin)
-----------------
- 4 StatCards: выручка сегодня (с трендом ▲▼% к вчера), выручка за период, издержки, чистый доход
- 3 ключевых метрики: средний чек, выручка на корт, маржинальность
- Выбор месяца (Янв–Июнь 2026) — общий для всех страниц
- Графики: выручка за месяц (линия), выручка по кортам (столбцы), загрузка % (горизонтальный)
- Сводка дня: всего / проведено / идут / отменено
- Кнопка «Сброс данных»

Сетка расписания (#/admin/schedule)
------------------------------------
- Строки = корты (4), колонки = часы (08:00–22:00)
- Навигация по дням ← сегодня →
- Цвет ячейки: серый (свободно), синий (запланировано), зелёный (идёт), тёмный (завершено)
- Клик по занятой ячейке → детали тренировки
- Клик по пустой → создание тренировки (выбор тренера + клиента)

Тренировки (#/admin/bookings)
-------------------------------
- Поиск с debounce 300ms
- Фильтры: период от/до, тренер, статус
- Сортировка по клику на заголовок (▲▼)
- Пагинация: 1–15 из N
- CSV-экспорт с учётом текущих фильтров
- Выбор месяца

Тренеры (#/admin/trainers)
----------------------------
- Таблица: имя, код, ставка, часов/мес, за месяц, всего, статус
- График зарплат за месяц
- Добавить тренера (модалка: имя, код, телефон, ставка, специализация)
- Редактировать / Отключить/Включить
- CSV-ведомость

Клиенты (#/admin/clients)
--------------------------
- Таблица: имя, телефон, абонемент, остаток, срок действия, всего тренировок

Корты (#/admin/courts)
-----------------------
- Графики: загрузка % (горизонтальный), выручка по кортам, пиковые часы
- Таблица: корт, тип, загрузка %, выручка, час пик, рекомендация
- Рекомендации: скидка на простои, акция в непиковые часы, повышение при 80%+

Финансы (#/admin/finance)
--------------------------
- 3 StatCards: выручка, издержки, чистый доход
- Круговой график структуры издержек
- P&L таблица по дням: выручка / издержки / доход

Реестр выплат (#/admin/registry)
---------------------------------
- Фильтры: месяц, тип операции (все/поступления/выплаты)
- Таблица: дата, тип (Приход/Расход), описание, сумма, статус (Проведено/Ожидает)
- Статистика обновляется под фильтр
- Данные из завершённых тренировок (выручка + оплата тренерам)


ТЕХНИЧЕСКИЕ ДЕТАЛИ
===================

Хранение: localStorage (ключ v9), авто-регенерация при обновлении
Данные: 6 тренеров, 5 клиентов, 4 корта, ~1500 броней (январь–июнь 2026)
Библиотеки: QRCode.js, jsQR, Chart.js 4.x, Font Awesome 6, Inter + Montserrat
Роутинг: SPA на хешах, EventBus для обновлений
Сессия: sessionStorage, переживает обновление страницы
Адаптив: клиент/тренер — мобильные (max 420px), админ — десктоп с сайдбаром
