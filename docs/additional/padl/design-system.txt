3. ИКОНОГРАФИЯ
---------------
  Библиотека:   Font Awesome 6 Free (Solid стиль)
  Размеры:      14px (в тексте), 18px (в кнопках), 24px (в карточках)
  Цвет:         наследует цвет текста

  Карта замены emoji → FA иконки:
    📊 = fa-chart-line       (аналитика)
    🛡️ = fa-shield-halved   (честный корт)
    💰 = fa-coins            (зарплаты)
    🎾 = fa-table-tennis-paddle-ball (падел)
    👤 = fa-user             (тренер)
    ⚙️ = fa-gear             (админ)
    🌿 = fa-leaf             (outdoor корт)
    🏠 = fa-warehouse        (indoor корт)
    📅 = fa-calendar-days    (дата)
    📋 = fa-clipboard-list   (история)
    🏟️ = fa-futbol          (корт)
    🎯 = fa-bullseye         (цель)

4. ОТСТУПЫ И РАЗМЕРЫ
---------------------
  Ширина сайдбара: 230px
  Высота хедера:   56px
  Радиус скругления: 10px (карточки), 6px (кнопки), 14px (мобильные карточки)
  Отступы: 4/8/12/16/20/24/32px

5. ТЕНИ
-------
  Карточка:    0 1px 3px rgba(0,0,0,0.06), 0 1px 2px rgba(0,0,0,0.04)
  Выпадающая:  0 4px 12px rgba(0,0,0,0.08)
  Модалка:     0 10px 40px rgba(0,0,0,0.12)
  Кнопка при наведении: поднимается на 2px

6. ПРАВИЛА
----------
  - НИКАКИХ эмодзи в интерфейсе. Только SVG-иконки.
  - Кнопки: скруглённые 6-12px, анимация scale при клике
  - Карточки: белые с лёгкой тенью, скруглённые
  - Статусы: цветные бейджи (синий/зелёный/красный/оранжевый/серый)
  - Пустые состояния: иконка 40px + текст + призыв к действию
  - Мобильный: max-width 420px, центрирован, bottom nav
  - Десктоп: сайдбар слева, светлый фон
