## 🚀 Как использовать прототипы

### Быстрый старт:
```bash
# 1. Клонировать репозиторий
git clone https://github.com/bestdeejay-design/arbat38.git
cd arbat38

# 2. Открыть любой прототип в браузере
open mobile-app.html
open airport-staff.html
open cashier.html
open restaurant-admin.html
```

### Сценарий тестирования:

**1. Сотрудник аэропорта создает ваучеры:**
- Откройте `airport-staff.html`
- Перейдите в "Создать ваучеры"
- Заполните форму (рейс SU 1234, 180 пассажиров, горячее питание)
- Нажмите "Создать ваучеры"
- Получите QR-код для стойки

**2. Пассажир получает ваучер:**
- Откройте `mobile-app.html`
- Перейдите на вкладку "Ваучер"
- Введите: SU 1234, 12A, Иванов
- Нажмите "Получить ваучер"
- Получите QR-код ваучера

**3. Кассир проверяет ваучер:**
- Откройте `cashier.html`
- Нажмите "Ввести вручную"
- Введите код: VCH-123456789
- Проверьте информацию
- Нажмите "Погасить"

**4. Пассажир делает предзаказ:**
- В `mobile-app.html` перейдите в "Рестораны"
- Выберите ресторан
- Добавьте блюда в корзину
- Перейдите в "Заказы"
- Оформите заказ (с применением ваучера)

**5. Менеджер ресторана управляет заказами:**
- Откройте `restaurant-admin.html`
- Видите новые заказы с анимацией
- Принимайте заказы
- Меняйте статусы
- Управляйте меню
